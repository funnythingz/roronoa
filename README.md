zorome is beautiful

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'roronoa'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roronoa

## Usage

default is `DateTime.now` scan.

```
% Roronoa::Zoro.eye
```

can use option

```
% Roronoa::Zoro.eye(2014,11,11,1,11)
```
